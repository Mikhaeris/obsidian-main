Answer:
	Time Complexity: O(N)
	Space Complexity: O(1)

Solution:
```cpp
class Solution {
public:
    int numberOfSpecialChars(string& word) {
        int lower = 0, upper = 0;

        for (const auto& ch : word) {
            if ('a' <= ch && ch <= 'z') {
                lower |= 1 << (ch - 'a');
            } else {
                upper |= 1 << (ch - 'A');
            }
        }

        int ans = 0;
        for (int i = 0; i < 26; ++i) {
            if ((1 << i & lower) && (1 << i & upper)) {
                ++ans;
            }
        }

        return ans;
    }
};
```
**Explanation:**
	Цель: Специальном символом считается сивол, который содержится в строке в верхнем и нижнем регистре. Нужно узнать сколько таких символов есть в строке.
	Решение: Сохранять проверку симвлов в верхнем и нижнем регистре в побитовой маске для, где i-й бит означет присутвие или отсутвие этого символа в строке. в конце пройтись по этим масскам и посчитать такие символы.
	P.S.  в данном случае можно хранаить и в массиве символов, но в побитовой маске меньше места. Т.е. 26 * 2 = 52 байта для двух массивов чаров или 4 * 2 = 8 байт для побитовой маске, которая хранится в двух переменных типа int.
