```cpp
class Solution {
public:
    int bestClosingTime(string& c) {
        int penalty = 0; 
        for (const auto& ch : c) {
            if (ch == 'Y') {
                ++penalty;
            }
        }

        int min_penalty = penalty;
        int hour = 0;
        for (int i = 0; i < c.size(); ++i) {
            if (c[i] == 'Y') {
                --penalty;
            } else {
                ++penalty;
            }
            if (min_penalty > penalty) {
                min_penalty = penalty;
                hour = i + 1;   
            }
        }
        return hour;
    }
};
```
**Explanation:**
	Цель: Дан массив, показывающий в какой час пришел или не пришел клиент. Нужно найти лучший час для закрытия магазина. При закрытии магазина считается штраф, если магазин открыт и в этот час никто не пришел, то штраф увеличивается на один, если магазин закрыт и пришел покупать, то штраф увеличивается на один.
	Pешение: Предположим, что магазин закроется в нулевой час, следовательно штраф будет равнятся пришедшим покупателям. Если идти и закрывать магазин в каждый последующий час, то штраф будет изменяться, если встретился покупатель, то штраф уменьшается, если никто не пришел, то штраф увелививается на один. Отслеживать текущий и минимальный штраф(с часом равным этому минимальному штрафу). Вернуть нужный час.
