**Complexity:** Easy
Answer:
	Time Complexity: O(N)
	Space Complexity: O(1)
Code:
Solution:
```cpp
class Solution {
public:
    int repeatedNTimes(vector<int>& nums) {
        int n = nums.size();
        for (int i = 2; i < n; ++i) {
            if (nums[i] == nums[i-1] || nums[i] == nums[i-2]) {
                return nums[i];
            }
        }
        return nums[0];
    }
};
```
**Explanation:**
	Цель: Дан массив. в нем хранятся числа, одно число содержится N/2 раз. Нужно найт такое число.
	Решение: Можно логически догадаться, что это похоже на мажорити элемент, но тут число повторяется половину массива. Значит оно повторяется и в какой-то момент предыдущее или прыдыдущее предыдущего будет равно текущему. Иначе вернуть первый элемент.
	P.S. по сути правильнее хранить в хэш таблице число и количество его повторений. И вернуть перый элемент, которого больше 1.
