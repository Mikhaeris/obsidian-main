**Complexity:** Easy
Answer:
	Time Complexity: O(N)
	Space Complexity: O(1)
Code:
Solution:
```cpp
class Solution {
public:
    int countTestedDevices(vector<int>& bP) {
        int count = 0;
        for (const auto& dev : bP) {
            if (dev - count > 0) {
                ++count;
            }
        }
        return count;
    }
};
```
**Explanation:**
	Цель: Дан массив устройств с зарядом батареи. Если заряд устройства больше 0, то увеличить количество тестируемых устройств и уменьшить заряд всех остальных устройств. Нужно узнать сколько устройств будет протестировано.
	Pешение: Чтобы пройтись за O(n) следует хранить сколько времени прошло и для каждого устройства вычитать это время из его заряда, чтобы узнать, что оно еще не разряжено. Но так как в задаче уже есть счетчик сохранения количества устройств, то можно использовать его как информцию для узнавания заряда устройства.
