```cpp
class Solution {
public:
    int countCollisions(string directions) {
        int n = directions.size();

        int l = 0;
        while (l < n && directions[l] == 'L') {
            ++l;
        }

        int r = n - 1;
        while (r >= l && directions[r] == 'R') {
            --r;
        }

        int ans = 0;
        for (int i = l; i <= r; ++i) {
            if (directions[i] != 'S') {
                ++ans;
            }
        }

        return ans;
    }
};
```
**Explanation:**
	Цель: Дана строка машин с их направлением движения. 'L'  - влево, 'R' - вправо, 'S' - стоит на месте. Нужно попределить сколько будте столкновений. Если сталкиваются движущиеся автомобили, то плюс два столкновения, если со стоящими, то плюс одно столкновение. Нужно определить сколько будет столкновений.
	Решение: Все сводится к тому, что нужно посчитать сколько есть автомобилей, которые движутся. Но перед этим нужно пропускать автомобили, которые точно не столкнутся - это автомобили которые стоялт слева и едут влевую строну и автмобили, которые стоят справа и движутся вправо.
