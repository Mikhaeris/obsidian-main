	Space Complexity: O(1)

Solution:
```cpp
class Solution {
public:
    int minimumOperations(vector<int>& nums) {
        int ans = 0;
        for (const auto& num : nums) {
            ans += (num % 3) ? 1 : 0;
        }
        return ans;
    }
};
```
**Explanation:**
	Цель: Дан массив чисел, нужно узнать сколько нужно совершить операций, чтобы получить число которое делится на три без остатка. Можно увеличвать или уменьшать число на один.
	Решение: Можно провести исследоание и увидеть, что числа, делящиеся на три без остатка находятся через два числа. Следовательно минимальное количество операций, чтобы сделать число делящимся на три без остатка, будет минимум равным одной. Значит нужно просто проверять, делится ли число на три и если не делится, то к ответу прибавлять один.
